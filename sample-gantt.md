# MarkMind 2.0 출시 로드맵

각 작업에 `@start(날짜)` / `@due(날짜)` / `@progress(0~100)` 마커를 붙이면
툴바에서 **간트 뷰**로 전환했을 때 막대 차트로 표시됩니다.

- `@start`만 있고 `@due`가 없으면 **마일스톤**(다이아몬드)으로 그려집니다.
- 진행률은 `@progress(...)` 또는 체크박스 `[x]`(100) / `[ ]`(0)로 표현합니다.
- **헤딩(H2)이 섹션(그룹)** 이 되고, 섹션마다 색이 자동 배정됩니다.

## 기획

- [x] 시장 조사 @start(2026-06-01) @due(2026-06-07) @progress(100)
- [x] 요구사항 정의 @start(2026-06-08) @due(2026-06-14) @progress(100)
- [x] 경쟁 제품 분석 @start(2026-06-09) @due(2026-06-13)
- 기획 확정 @start(2026-06-15)
