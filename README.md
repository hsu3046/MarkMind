# MarkMind

## Tagline-en

MarkMind renders AI-generated Markdown documents into a beautifully structured, highly readable format. Integrate all your various AI document workflows into a single place with MarkMind, and manage everything from translation and summarization to editing easily and smartly with the power of AI.

## Summary-en

MarkMind consolidates scattered, unstructured dataвҖ”such as handwritten notes, voice recordings, and scanned PDFsвҖ”into a highly readable document environment. It instantly transforms large media assets, including audio recordings up to 4 hours long, into editable text data. Through seamless inline AI editing, tasks like translation, summarization, and proofreading are handled in real time within the same editor, ensuring an uninterrupted workflow. In addition to providing a macOS-optimized experience, all user-specific security credentials for AI integration are encrypted and stored inside the native macOS Keychain, ensuring absolute peace of mind in security-critical business and professional environments.

## вңЁ What It Does

### AI Agent (вҢҳI)

One side panel with eight modes вҖ” speech recognition, image OCR, meeting notes, slides, image generation, plus three text-editing modes. Switch the AI model **inline per mode** (a dropdown that shows only available key/subscription models, each with its company logo), or set the default in Settings (company / auth / model).

- **Fixes grammar in-place** вҖ” Select text, pick "л¬ёлІ• көҗм •," and see word-level highlights of every correction вҖ” accept or reject one by one.
- **Translates seamlessly** вҖ” Korean вҶ” English вҶ” Japanese with paragraph-by-paragraph before/after diffs.
- **Improves entire documents** вҖ” Free-form prompt with Gemini 3.1 Pro for deep rewrites.
- **Generates meeting notes** вҖ” Full document as transcript вҶ’ structured Markdown notes via Claude Sonnet 4.6 or Gemini 3.1 Pro. Built-in templates (general / detailed / team-sync) + user-defined templates.
- **Generates slide decks** вҖ” Export the current document as a `.pptx` with an AI-planned layout (Claude or Gemini).
- **Generates images** вҖ” Text-to-image (plus reference images) via Gemini (Nano Banana), ChatGPT (GPT Image 2), or Grok (Grok Imagine); insert into the document or save to file. Works with an API key **or a subscription**.
- **Pops up an AI bar on selection** вҖ” Highlight any text in the editor and a floating action bar appears instantly.
- **Streams AI responses live** вҖ” Watch the rewrite appear in real time, then review the smart diff.

### Multimodal Input (doc-converter integration)

- **Speech вҶ’ Text (мқҢм„ұ мқёмӢқ)** вҖ” Drop an audio file (up to 4h / 1GB). Auto-chunked, speaker-labeled transcription with **Silero VAD** silence trimming (saves ~30вҖ“40% Gemini cost). Original recording timestamps preserved across chunks.
- **Image вҶ’ Text (мқҙлҜём§Җ мқёмӢқ)** вҖ” 2-pass OCR (Gemini Flash extraction вҶ’ Gemini Pro refinement) for handwriting, screenshots, and PDFs up to 250MB. Auto-fallback to pdfium rasterization for stubborn PDFs.
- **Inline OCR drag-drop** вҖ” Drop an image directly into the editor (when no sidebar active) вҶ’ OCR result inserted at cursor.
- **Sidebar drag-drop** вҖ” Drop audio/PDF/image files into the active sidebar вҶ’ file auto-attached by extension.
- **Speaker diarization (нҷ”мһҗ л¶„лҰ¬)** вҖ” Verified **pyannote**, two ways: local (free/offline `pyannote.audio` Python sidecar) or cloud (pyannote.ai API). Post-transcription speaker rename/merge; falls back to Gemini's own speaker guesses if neither is configured.

### Visualization

- **Mindmap (вҢҳ4)** вҖ” Render the document's heading/bullet hierarchy as an interactive node graph; edit nodes to update the document.
- **Flowchart (вҢҳ5)** вҖ” AI generates a BPMN-lite flow diagram from the document or a topic (an empty-state guide is shown before generation).
- **Gantt (вҢҳ6)** вҖ” AI breaks the document or a topic into a scheduled Gantt chart.
- **AI generation (auto-generate modal)** вҖ” Flowchart/Gantt/Mindmap each open a generate modal: pick the **source** (*auto-analyze* the current document, or *direct input* of a topic вҖ” the current document is still passed as background context) and the **write mode** (*append to* or *replace* the document).

### Editor Experience

- **Renames files inline** вҖ” Click the filename in the toolbar to edit. Saved files renamed on disk automatically.
- **Renders beautiful previews** вҖ” Side-by-side split, full-width reading mode, synchronized scroll, code syntax highlighting, frontmatter rendered as metadata box.
- **Navigates with outline** вҖ” Auto-extracted H1вҖ“H6 headings, click to jump.
- **Opens multiple windows** вҖ” Cascade-positioned, `вҢҳN` shortcut, recent files in File menu, Finder file association for `.md`.
- **Searches everywhere** вҖ” CodeMirror search in the editor (Korean UI, `вҢҳF`) plus DOM-based highlight in preview mode.
- **Switches themes instantly** вҖ” Dark and light with Pretendard CJK-optimized typography.
- **Custom editor background** вҖ” Pick a custom background color per preference (BackgroundPicker), preserved across light/dark.
- **Rich table editing** вҖ” Table bubble menu, merged-cell rendering, and column-width control for Markdown tables.

### Security & Settings

- **Unified Settings modal** вҖ” One place for Gemini, Claude, OpenAI, and Grok (xAI) keys (File вҶ’ Settings).
- **macOS Keychain storage** вҖ” All API keys live in the OS Keychain (`space.knowai.markmind`), never in localStorage on the desktop app.
- **Legacy migration** вҖ” First launch auto-migrates any old `localStorage` keys to Keychain.
- **Subscription OAuth** вҖ” Reuse an existing AI subscription by reading the local CLI login вҖ” no API key needed: **Claude** (Max/Pro via `claude`), **ChatGPT** (Plus via `codex`, text + image), **Gemini** (via the `agy` Antigravity CLI), **Grok** (via `grok login`; note the general API needs a paid SuperGrok plan). Used for both text and image generation.

### Claude Integration (MCP) вҖ” new in 0.4.0

- **Reads & edits your open documents from Claude** вҖ” An in-process **MCP server** (Streamable HTTP, `127.0.0.1:8417`) lets Claude Desktop / Claude Code see the documents you have open (live editor content, including unsaved changes) and modify them.
- **Read tools** вҖ” list open documents, current (focused) document, heading outline, the user's current text selection.
- **Edit tools** вҖ” surgical `str_replace`, full rewrite, replace selection, insert at cursor/end. Changes apply to the **live editor and are left unsaved** (you save) вҖ” preserving undo where possible.
- **`propose_edit`** вҖ” Claude proposes a rewrite shown as a **diff you must accept or reject**; nothing changes unless you accept.
- **`create_document` / `save_document`** вҖ” open a Claude-drafted document in a new window; persist to disk on request.
- **Local & isolated** вҖ” binds `127.0.0.1` only; one-line setup via `mcp-remote` (see in-app Tutorial В§ 7).

### Cloud Sync & Export

- **Google Drive sync** вҖ” Bring-your-own OAuth client; saved files auto-upload to a `MarkMind` Drive folder, and you can browse/open documents from Drive across machines.
- **PDF export** вҖ” Native macOS WKWebView print pipeline (`NSPrintInfo`) вҶ’ PDF of the rendered document.

## рҹ“Ұ Installation

### macOS (Native App)

Download the latest `.dmg` from [Releases](https://github.com/yuhitomi/markmind/releases).

First launch:
1. Open **File вҶ’ Settings** to enter at least one API key:
   - **Gemini** (required for speech/OCR/grammar/translate/improve): [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
   - **Claude** (optional, default for meeting notes): [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys)
   - **OpenAI** (optional): [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. macOS Keychain may prompt вҖ” choose **"Always Allow"** to avoid future prompts for the same build.
3. (Audio first run only) ffmpeg auto-downloads to user cache вҖ” takes ~30s on first transcription.

### Build from Source

```bash
# Prerequisites: Node.js 18+, Rust 1.75+, Xcode CLI Tools

git clone https://github.com/yuhitomi/markmind.git
cd markmind
npm install

# Speaker diarization (optional): verified pyannote, two ways вҖ”
#   вҖў Local (free/offline): set MARKMIND_DIAR_PYTHON to a Python with `pyannote.audio`
#     installed (+ a free HuggingFace token, accept the gated models once).
#   вҖў Cloud (paid): enter a pyannote.ai API key in Settings.
# Without either, transcription falls back to Gemini's own speaker guesses.

# Development (HMR enabled)
npm run tauri dev

# Production build (.app + .dmg in src-tauri/target/release/bundle/)
npm run tauri build

# Run Rust unit tests (VAD post-process, timestamp mapping, etc.)
cargo test --manifest-path src-tauri/Cargo.toml --lib
```

### Web Only (no AI sidebars)

```bash
npm run dev
# Open http://localhost:1420
```
